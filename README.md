# chrome-extension
### Tip
  -  name.crx  

         插件名称.crx
        
  - name.crx => name.[rar|zip|...]  

        手动查看 name.crx 是否存在危险代码
        将 name.crx 后缀名 改成 rar | zip | 其他压缩格式 , 解压查看源码即可 !
        
  - 安装方法
    
        1. chrome ->  更多工具 -> 扩展程序 -> 将 插件 拖拽进 拖拽以安装 中
        2. chrome ->  更多工具 -> 扩展程序 -> 加载已解压的扩展程序 (文件夹形式,插件解压的文件)
        
  - 目录说明
        
        一级文件夹 -> 二级文件夹  -> 文件
        分类         详细分类       文件
        若没有 二级文件夹 文件名中的 数字 代表 优先推荐哦 !
